# Create a page button on detail layout

{% hint style="info" %}
You can control the **Edit and Delete** button on a detail layout page if users have [permissions](https://docs.stackerhq.com/stacker/ai-and-automations/action-buttons/broken-reference) to Edit or Delete records from **Page Buttons.**
{% endhint %}
