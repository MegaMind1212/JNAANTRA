# Agent Triggers

{% hint style="info" %}
This feature is currently in Beta. Please reach out to Support to find out more.&#x20;
{% endhint %}

AI agents can be triggered by workflows, emails or through a chat on your detail pages.

* App Chatbot
* Email Recieved
* External Chatbot
* Record Created
* Recurring Schedule
